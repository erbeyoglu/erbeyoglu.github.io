Slide 28: by n = 5 both rows of Pⁿ are (0.75, 0.25) — the probability of rain five days ahead no longer depends on today. The Markov property (slide 10) only ever looks one step back; iterating it washes out even that. The limiting values (0.75, 0.25) become Week 13’s steady-state story.

Where does P² come from? Slides 22–33

Chapman-Kolmogorov is just “sum over every stopover”. Pick a start, a destination, and the number of steps — every path is listed with its probability product, and their sum is exactly the Pⁿ entry.

2

Slide 23 in action: P(X₃ = A | X₁ = B) = Σₖ p(B→k)·p(k→A) = 0.14 — the (B, A) entry of P². Matrix multiplication IS path enumeration; that is all Chapman-Kolmogorov says.

Transient: some escape route never comes back (slide 42). Recurrent: return is certain (slide 43). Absorbing: pᵢᵢ = 1 — recurrent with no exit at all (slide 45). These are class properties: states that communicate share their fate (slide 46). Recurrent + aperiodic = ergodic (slide 50).

Build the insurance company’s matrix Slide 52

Premiums depend on the last TWO years, so the state must remember both: (year before, last year) with N = no accident, Y = accident. Accident probability this year: 10% if there was one last year, 3% otherwise. Fill in all 16 entries.

from \\ toNNNYYNYY

spoiler — open after the class has answeredThe trick: next year’s state is (last year, this year) — the oldest year falls off. From NN you can only reach NN or NY; a row like NN→YY must be 0 because the “year before” of the next state is forced. Premiums (250/400/800) attach to states — the long-run average premium is Week 13’s steady-state story.
