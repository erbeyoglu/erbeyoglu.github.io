Produce now, roll the dice Slides 13–24

Three periods, demand each period is 1 or 2 with probability ½ each; production cost 2x + 3 (if x > 0), holding $1, end-of-period inventory ≤ 3, all demand met on time; leftovers after period 3 sell for $2. Start with 1 unit. First reveal the optimal policy, then play it against random demand and watch the average converge to the DP value.

The state after a decision is now RANDOM: producing x from inventory i leads to i + x − 1 or i + x − 2, each with probability ½ — so the recursion averages the two futures (slide 20). An optimal solution is no longer a fixed plan but a policy: at stage 1 produce 3, then react to whatever demand materializes (slide 24).

Stay in line or give up? Slides 26–33

Hit-and-Miss Manufacturing Slides 34–45

One acceptable item is needed; each produced item is acceptable with probability ½. Setup $300 + $100/item per run, at most 3 runs, and failing all runs costs $1600. Choose the lot size for each run (applies only if all previous runs failed) and minimize the expected cost. Beat $675!

State = do we still need an item (1) or not (0); P(a lot of x yields none acceptable) = 0.5ˣ (slides 37–41). Expected cost = run 1 + 0.5^x₁·[run 2 + 0.5^x₂·(run 3 + 0.5^x₃·1600)].
