Warm-up: does f(x) = 48x − 60x² + x³ have a global max? Slide 3

Vote first: global max — yes or no? Then find both local extrema (you’ll need to zoom) and watch what the tails do.

f′(x) = 48 − 120x + 3x² = 0 gives x ≈ 0.404 (f″ < 0: local max, f ≈ 9.68) and x ≈ 39.60 (f″ > 0: local min, f ≈ −32036). But x³ dominates: f → +∞ as x → +∞ and −∞ as x → −∞, so there is no global max or min — local analysis alone can’t certify global optimality.
